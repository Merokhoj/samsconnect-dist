# SamsConnect

A powerful, high-performance Android & iOS device management studio built with Flutter.

## Features

- **Device Dashboard**: Real-time hardware monitoring (CPU, RAM, Battery, Storage).
- **Screen Mirroring**: Ultra-low latency screen mirroring for Android (via scrcpy) and iOS (via UxPlay).
- **File Studio**: Robust file browser with drag-and-drop file transfer support.
- **App Management**: Browse, install, and uninstall applications.
- **Advanced Tools**: Unlock patterns, fetch device properties, and high-resolution screenshotting.

## Getting Started

### Prerequisites

- **FFmpeg**: Required for screen mirroring.
- **libimobiledevice & ifuse**: Required for iOS connection.
- **ADB**: Bundled (extracted on first launch).

### Running

```bash
flutter run
```
